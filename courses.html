Follow these steps to decode the encrypted video files:

  • 1
    Download Files Download the decoder files and videos using the download buttons. Place all files in the same folder.
  • 2
    Install Node.js Make sure to have Node.js installed. Download from nodejs.org
  • 3
    Run setup.bat Double-click setup.bat to install FFmpeg and required dependencies.
  • 4
    Decode Videos Open Command Prompt in the folder and run:
    node decoder.js output_encoded1.mp4
    Replace with output_encoded2.mp4 or output_encoded3.mp4 as needed.
